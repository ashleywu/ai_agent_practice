# 快速设置新仓库：ahacatcher

## 方法 1: 使用脚本（推荐）

### 步骤 1: 在 GitHub 创建新仓库
1. 访问：https://github.com/new
2. 仓库名：`ahacatcher`
3. **不要**勾选 "Initialize with README"
4. 点击 **Create repository**

### 步骤 2: 运行设置脚本
```powershell
cd c:\Users\peili\fastapi_hello\phaseBp1
.\setup_new_repo.ps1
```

脚本会自动：
- 初始化 git（如果需要）
- 添加所有文件
- 创建提交
- 添加新仓库为 remote
- 推送到 GitHub

## 方法 2: 手动执行（如果脚本有问题）

### 步骤 1: 在 GitHub 创建新仓库
同上

### 步骤 2: 在 phaseBp1 目录执行

```powershell
cd c:\Users\peili\fastapi_hello\phaseBp1

# 如果还没有 git 仓库，初始化
git init

# 添加所有文件
git add .

# 创建提交（如果还没有）
git commit -m "Initial commit: Aha! Catcher"

# 添加新仓库为 remote
git remote add origin https://github.com/ashleywu/ahacatcher.git

# 如果已经有 origin，先删除再添加
# git remote remove origin
# git remote add origin https://github.com/ashleywu/ahacatcher.git

# 推送到新仓库
git branch -M main
git push -u origin main
```

## 步骤 3: 更新部署配置

推送成功后，更新 `deploy.py` 使用新仓库：

```powershell
# 部署时使用新仓库 URL
cd c:\Users\peili\fastapi_hello\phaseBp1
python deploy.py --repo-url https://github.com/ashleywu/ahacatcher.git --service-name pw-aha-catcher --branch main
```

或者修改 `deploy.py` 的默认值（可选）。
