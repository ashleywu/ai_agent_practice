# Rename Vercel Project to practicechat

## Steps to Get `practicechat.vercel.app` URL

### Step 1: Rename Project in Vercel Dashboard

1. **Go to**: https://vercel.com/dashboard
2. **Click** on your project: `ai-agent-practice`
3. **Go to**: **Settings** → **General**
4. **Find**: **Project Name** field
5. **Change it to**: `practicechat`
6. **Click**: **Save**

### Step 2: Wait for Update

- Vercel will automatically update your URL
- Your new URL will be: **`https://practicechat.vercel.app`**
- This usually takes a few seconds

### Step 3: Verify

- Check your project dashboard - the URL should update automatically
- The old URL will redirect to the new one

## Important Notes

- ✅ The project name can contain lowercase letters, numbers, and hyphens
- ✅ `practicechat` is a valid project name
- ✅ Your new URL will be: `https://practicechat.vercel.app`
- ⚠️ Old URL will still work but redirect to new one
- ⚠️ If `practicechat` is taken, try `practice-chat` or `practicechat-app`

## Alternative: If Name is Taken

If `practicechat` is already taken by another Vercel user, try:
- `practice-chat`
- `practicechat-app`
- `practicechat-ai`
- `chat-practice`

The URL will be: `https://[your-chosen-name].vercel.app`
